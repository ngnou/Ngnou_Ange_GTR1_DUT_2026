Introduction

Ce mini site a été créé dans le cadre du cours de TP programmation

Il permet de découvrir la création de plusieurs pages HTML reliées entre elles.

Objectif du mini-site

L'objectif est d'apprendre à structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page Acceuil présente le contexte général du mini-site.
  • La page présentation contient les informations personnels de l'étudiant.
  • La page Contact contient les informations de contact de l'étudiant.
